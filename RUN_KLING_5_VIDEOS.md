# 🎬 5개 영상 자동 생성 (EP1 4개 + EP2 1개)

## 📋 개요

이 가이드는 **EP1 전체 + EP2 첫 번째 영상**을 Kling.ai에서 자동으로 생성하는 방법입니다.
- **생성할 영상:** 5개
- **예상 시간:** 15~25분 (약 20분)

---

## 🚀 빠른 시작 (2단계)

### 1단계: 필수 패키지 설치 (처음 한 번만)

```bash
pip install selenium webdriver-manager
```

### 2단계: 스크립트 실행

```bash
python3 scripts/run_kling_5_videos.py
```

또는

```bash
python run_kling_5_videos.py
```

---

## 📊 생성할 5개 영상

| # | 에피소드 | 씬 | 제목 | 내용 |
|---|---------|-----|------|------|
| 1 | **EP1** | 1-1 | 정혼식 준비 | 불안한 인어왕자의 정혼식 준비 |
| 2 | **EP1** | 1-2 | 인간여자 구조 | 해변에서 인간여자를 구출하는 순간 |
| 3 | **EP1** | 1-3 | 왕궁으로 돌아옴 | 정약혼자 공주와의 어색한 만남 |
| 4 | **EP1** | 1-4 | 몰래 해변 방문 | 밤 해변에서 비밀 재만남 |
| 5 | **EP2** | 2-1 | 해변에서의 비밀 만남 | 황혼의 해변에서 행복한 로맨스 |

## 🎯 실행 과정

### 1. 스크립트 실행

```
python3 scripts/run_kling_5_videos.py
```

**출력:**
```
======================================================================
🎬 인어왕자 - Kling.ai 5개 영상 자동 생성
======================================================================

📝 생성할 영상:
   • EP1 씬 1-1: 정혼식 준비
   • EP1 씬 1-2: 인간여자 구조
   • EP1 씬 1-3: 왕궁으로 돌아옴
   • EP1 씬 1-4: 몰래 해변 방문
   • EP2 씬 2-1: 해변에서의 비밀 만남

⏱️  예상 소요 시간: 15~25분 (약 20분)

시작하시겠습니까? (y/n): 
```

### 2. 시작 확인

터미널에서 `y` 입력 후 **엔터** 누르기.

### 3. Chrome 자동 열림

```
✅ Chrome 브라우저 시작됨

🌐 Kling.ai 접속 중...
✅ Kling.ai 접속 완료

🔐 Kling.ai에 로그인해주세요.
   (로그인 창이 열렸습니다. 300초 동안 로그인을 기다리고 있습니다.)
```

### 4. 수동 로그인

**당신이 할 일:**
- Chrome 브라우저에서 Kling.ai에 **수동으로 로그인** (이메일, Google 등)
- 로그인 완료 후 터미널로 돌아가기
- 터미널에서 **엔터** 누르기

```
로그인을 완료한 후 엔터를 누르세요...
```

### 5. 자동 생성 시작

로그인 후 스크립트가 자동으로 5개 영상을 순차 생성합니다:

```
============================================================
[1/5] EP1 - 씬 1-1: 정혼식 준비
============================================================
✅ 프롬프트 입력 완료 (길이: 250자)
✅ 생성 버튼 클릭 완료

⏳ 영상 생성 중... (타임아웃: 600초)
   진행 중... (30초 경과)
   진행 중... (60초 경과)
   ...
✅ 영상 생성 완료! (소요 시간: 145초)

⏳ 다음 프롬프트 처리 준비 중... (3초 대기)

============================================================
[2/5] EP1 - 씬 1-2: 인간여자 구조
============================================================
...
```

### 6. 최종 결과 요약

모든 5개 영상 생성 완료 후:

```

======================================================================

❌ 실패: 0/5
⏱️  총 소요 시간: 625초 (10분 25초)

에피소드 씬           상태           소요 시간      
--------------------------------------------------

EP1      씬 1-2     ✅ COMPLETED       142초
EP1      씬 1-3     ✅ COMPLETED       138초
```

## 🐛 문제 해결

### 문제 1: "프롬프트 입력 필드를 찾지 못했습니다"

**원인:** Kling.ai UI 변경 또는 페이지 로드 지연

**해결책:**
1. 브라우저에서 수동으로 프롬프트 입력
2. 엔터 키를 눌러 계속 진행

### 문제 2: "생성 버튼을 찾지 못했습니다"

**원인:** 버튼 UI 변경 또는 로그인 만료

**해결책:**
1. 브라우저에서 수동으로 생성 버튼 클릭
2. 엔터 키로 계속 진행

### 문제 3: "생성 타임아웃"

**원인:** Kling.ai 서버 과부하

**해결책:**
1. "계속 기다리기 (y)" 선택 → 추가 5분 대기
2. 또는 "n" 선택 → 다음 프롬프트로 진행

### 문제 4: Python 오류

```bash
ModuleNotFoundError: No module named 'selenium'
```

**해결책:**
```bash
pip install --upgrade selenium webdriver-manager
```
