# 🎬 Kling.ai 영상 생성 자동화 가이드

## 📋 개요

이 가이드는 Kling.ai에서 프롬프트 기반 AI 영상을 자동으로 생성하는 방법을 설명합니다.

---

## 🚀 빠른 시작

### 1단계: 필수 패키지 설치

```bash
pip install selenium webdriver-manager
```

### 2단계: 스크립트 실행

```bash
python3 scripts/kling_video_generator.py
```

### 3단계: 프롬프트 입력

스크립트가 시작되면 프롬프트를 한 줄씩 입력합니다:

```
프롬프트를 입력하세요 (한 줄씩, 빈 줄로 종료):
깊은 바다의 산호 궁전, 푸른 물빛이 들어오는 조명...
밤의 해변 근처 바다. 파도가 격렬하게 치고 있고...
...
(빈 줄 입력하여 종료)
```

### 4단계: 로그인

Chrome 브라우저가 자동으로 열리고 Kling.ai로 이동합니다.

**중요:** Kling.ai에 수동으로 로그인해주세요.

```
🔐 Kling.ai에 로그인해주세요.
   (로그인 창이 열렸습니다. 300초 동안 로그인을 기다리고 있습니다.)
```

로그인 완료 후 엔터를 누르세요.

### 5단계: 자동 생성

프롬프트가 자동으로 입력되고 영상이 순차적으로 생성됩니다:

```
============================================================
[1/20] 프롬프트 처리 중
============================================================
✅ 프롬프트 입력 완료 (길이: 150자)
✅ 생성 버튼 클릭 완료

⏳ 영상 생성 중... (타임아웃: 600초)
   진행 중... (30초 경과)
   진행 중... (60초 경과)
...
✅ 영상 생성 완료! (소요 시간: 145초)

[2/20] 프롬프트 처리 중
...
```

### 6단계: 결과 확인

모든 프롬프트 처리 완료 후 요약이 표시됩니다:

```
============================================================
📊 처리 완료 요약
============================================================

✅ 완료: 20/20
❌ 실패: 0/20
⏱️  총 소요 시간: 2900초 (48분 20초)

프롬프트          상태           소요 시간      
----------------------------------------
[ 1]        ✅ COMPLETED         145초
[ 2]        ✅ COMPLETED         138초
[ 3]        ✅ COMPLETED         152초
...
```

## 💡 사용 팁

### 프롬프트 입력 방식

**방식 1: 대화형 입력**
```bash
python3 scripts/kling_video_generator.py
# 대화형으로 프롬프트 입력
```

**방식 2: 파일에서 입력 (향후)**
```bash
python3 scripts/kling_video_generator.py --file prompts.txt
```

**방식 3: 커맨드라인 인자 (향후)**
```bash
python3 scripts/kling_video_generator.py "프롬프트1" "프롬프트2" "프롬프트3"
```

### 생성 시간 예상

각 프롬프트당 약 **2~5분** 소요됩니다.

**예시 계산:**
- 프롬프트 개수: 20개
- 평균 생성 시간: 3분/개
- 총 소요 시간: 60분

### 안정성 팁

1. **인터넷 연결 확인**
   - 안정적인 WiFi/유선 연결 필수
   - 생성 중 연결 끊김 방지

2. **브라우저 모니터링**
   - Chrome 창을 최소화하지 않기 (완전 종료도 금지)
   - 브라우저 창 크기 변경 가능

3. **시간 충분히 할당**
   - 장시간 작업이므로 충분한 여유 시간 확보
   - 노트북 절전 모드 비활성화

4. **Kling.ai 크레딧 확인**
   - 영상 생성에 크레딧 필요
   - 사전에 충분한 크레딧 확보

## 🐛 문제 해결

### 문제 1: "프롬프트 입력 필드를 찾지 못했습니다"

**원인:** Kling.ai UI 변경 또는 페이지 로드 지연

**해결책:**
1. 브라우저에서 수동으로 프롬프트 입력
2. 엔터 키를 눌러 계속 진행
3. 스크립트가 자동으로 적응

### 문제 2: "생성 버튼을 찾지 못했습니다"

**원인:** 버튼 UI 변경 또는 로그인 만료

**해결책:**
1. Kling.ai에 다시 로그인
2. 브라우저에서 수동으로 생성 버튼 클릭
3. 엔터 키로 계속 진행

### 문제 3: "생성 타임아웃"

**원인:** Kling.ai 서버 과부하 또는 네트워크 지연

**해결책:**
1. 메시지에서 "계속 기다리기"(y) 선택
2. 추가 5분 대기
3. 여전히 완료되지 않으면 다음 프롬프트로 진행(n)

### 문제 4: 로그인 후에도 인식되지 않음

**원인:** 자동화 감지 또는 세션 문제

**해결책:**
1. 로그인 완료 후 잠깐 대기 (3~5초)
2. 엔터 키 누르기
3. 페이지가 로드될 때까지 기다리기

## 📊 결과 확인

### 생성된 영상 위치

모든 생성된 영상은 **Kling.ai 내 대시보드**에 자동으로 저장됩니다.

```
Kling.ai 대시보드
├── My Videos (내 영상)
│   ├── [프롬프트 1] (시간)
│   ├── [프롬프트 2] (시간)
│   └── ... (20개)
└── 다운로드 가능
```

### 영상 다운로드

1. Kling.ai 대시보드에서 각 영상 선택
2. "Download" 버튼 클릭
3. 로컬 컴퓨터에 저장
