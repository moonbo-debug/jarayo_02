# 비디오 프롬프트 생성 스킬 (Video Prompt Generator)

## 개요

스토리 아키텍트가 제공한 각 에피소드의 가이드를 받아, **영상 생성 AI에 최적화된 4개의 15초 프롬프트**를 작성합니다. 각 프롬프트는 시작(오프닝) → 전개(액션/감정) → 종료(연결)의 명확한 구조를 가져야 하며, 전체 5부작에서 캐릭터와 배경의 일관성을 유지합니다.

## 핵심 원칙

### 1. 15초 분량 최적화

**15초의 구조:**
```
오프닝 (0~5초): 씬의 배경과 등장인물 소개
           ↓
전개 (5~12초): 감정, 액션, 상황 변화
           ↓
종료 (12~15초): 다음 씬으로의 연결, 감정 여운
```

**작성 규칙:**
- 과도한 액션이나 급격한 장면 전환 금지 (15초는 짧음)
- 한 가지 감정이나 상황에 집중
- 마지막 프레임이 다음 씬의 오프닝과 자연스럽게 연결

### 2. 영상 생성 AI 맞춤형 작성

**프롬프트 구조 (Recommended Format):**

```
[배경/환경] [조명/분위기] [등장인물] [감정/액션] [카메라 움직임] [색감] [음향 톤]
```

**예시:**
```
"깊은 바다의 산호 궁전, 푸른 물빛이 들어오는 조명, 
인어왕자가 어두운 표정으로 창문을 통해 바다를 바라봄, 
카메라는 천천히 옆으로 팬, 
색감은 차갑고 고요함, 
음향은 서정적인 현악기 배경음악"
```

**피해야 할 표현:**
- 부정형: "~하지 않는", "~없는" (대신 "~인", "~으로 가득한" 사용)
- 모호한 표현: "아름다운", "좋은" (대신 "반짝이는", "우아한" 등 구체적 형용사)
- 과도한 명령: "꼭 이렇게" (대신 제안 형태로)

### 3. 캐릭터/배경 일관성

**캐릭터 정의 및 유지:**
```
인어왕자:
  - 외형: 밝은 파란 비늘, 긴 흰색 머리, 우아한 체형
  - 성격: 민감하고 로맨틱함, 나중에 결단력 있음
  - 의상: EP1~2는 화려한 궁전복, EP3~4는 천박한 옷, EP5는 인간 의상

인간여성:
  - 외형: 갈색 머리, 밝은 눈, 수줍음
  - 성격: 진심 있고 강함
  - 의상: 항상 일관된 여름 옷 (해변 배경)

다른 인어공주:
  - 외형: 어두운 검은 비늘, 검은 머리, 위압적 체형
  - 성격: 권력욕 있고 집착함
```

**배경 정의:**
```
인어 궁전: 산호, 보석, 밝은 파란 물빛
바다 심해: 어둡고 신비로운, 초록-검은 물빛
해변: 밝고 따뜻한 황금빛 햇빛
```

### 4. 감정 표현 (Emotional Tone)

**프롬프트에 명시할 감정 상태:**
- 불안, 기대, 설렘, 두려움, 충격, 절망, 회복, 결연함

**시각적 감정 표현:**
| 감정 | 색감 | 조명 | 속도 |
|------|------|------|------|
| 불안 | 어두운 파랑, 검은색 | 불규칙한 그림자 | 빠름 |
| 설렘 | 밝은 황금, 핑크 | 따뜻하고 고르게 | 중간 |
| 절망 | 차갑고 흐린 회색 | 극적인 역광 | 느림 |
| 희망 | 밝은 파랑, 주황 | 밝은 햇빛 | 중간 |

## 프롬프트 작성 템플릿

### 마크다운 포맷

```markdown
## EP{N}: {에피소드 제목}

### 씬 {N}-{N} (0~15초): {씬 제목}

**프롬프트 (영상 생성 AI용):**
[구체적이고 생생한 프롬프트 150~200자]

**분석:**
- **배경:** ...
- **등장인물:** ...
- **감정:** ...
- **카메라:** ...
- **색감:** ...
- **음향:** ...

**연결점:** 다음 씬으로 어떻게 자연스럽게 이어지는지

---

### 씬 {N}-{N} (15~30초): {씬 제목}
...
```

### 실제 예시

```markdown
## EP1: 불운한 정혼

### 씬 1-1 (0~15초): 왕자의 불안

**프롬프트:**
"반짝이는 산호로 장식된 인어 궁전의 광장, 푸른 물빛이 들어오는 창문을 통해 은빛 햇빛이 내려옴. 
긴 흰색 머리를 한 인어왕자가 화려한 파란 비늘 복장으로 단정히 서 있음. 
하지만 표정은 걱정과 불안으로 가득함. 
카메라는 천천히 앞으로 이동하여 왕자의 불안한 얼굴을 클로즈업. 
색감은 우아하면서도 차갑고 어두운 톤. 
배경 음악은 섬세하고 불안한 현악기 선율."

**분석:**
- 배경: 인어 궁전 광장 (화려함)
- 등장인물: 인어왕자 (불안한 심정)
- 감정: 불안감 속의 우아함
- 카메라: 천천히 전진하여 클로즈업
- 색감: 파랑 + 은빛 + 어두운 톤
- 음향: 현악기의 불안한 멜로디

**연결점:** 왕자의 불안한 표정으로 종료 → 다음 씬에서 정혼식 준비로 이어짐
```

## 주의사항

### 캐릭터 일관성 체크리스트

각 프롬프트 작성 후:
- [ ] 캐릭터의 외형이 이전 씬과 동일한가?
- [ ] 캐릭터의 성격/감정 상태가 스토리와 일치하는가?
- [ ] 의상이 적절한가? (EP 진행에 따라 변화해야 함)
- [ ] 배경이 이야기상 맞는 위치인가?

### 시각 일관성 체크리스트

각 에피소드의 4개 프롬프트 완성 후:
- [ ] 색감 팔레트가 에피소드 전체에서 일관성 있는가?
- [ ] 카메라 이동이 자연스럽게 연결되는가?
- [ ] 배경 물리 법칙이 지켜지는가? (물이 흐르는 방향, 빛의 각도 등)
- [ ] 음향 톤이 에피소드의 감정 곡선과 일치하는가?

## 트리거 조건

이 스킬이 활성화되는 경우:
1. 스토리 아키텍트가 에피소드 분석 완료 후 각 에이전트에 작업 할당
2. 프롬프트 크리에이터가 "EP{N} 프롬프트 생성해줘" 요청
3. 기존 프롬프트 수정: "EP{N} 씬 {N}-{N}의 톤을 더 ~하게 해줘"

## 재호출 시 행동

이전 프롬프트가 있으면:
1. 기존 프롬프트 파일 확인
2. 사용자 피드백에 따라 해당 부분만 수정 (전체 재작성 X)
3. 수정 내용을 명확히 기록

## 테스트 시나리오

**테스트 1: 단일 에피소드 프롬프트 생성**
입력: EP1 스토리 가이드 (불안한 정혼, 인간여자 구조)
기대: 4개의 프롬프트 생성, 각각 15초 분량, 캐릭터/배경 일관성 유지

**테스트 2: 감정 곡선 반영**
입력: EP3 스토리 가이드 (갈등 폭발, 왕자 도망)
기대: 불안 → 긴장 → 충격 → 절망의 감정 변화를 프롬프트에 반영

**테스트 3: 캐릭터 진화**
입력: EP1, EP3, EP5 프롬프트 비교
기대: 왕자의 외형/의상/감정이 에피소드 진행에 따라 자연스럽게 변화
