# Kling.ai 영상 생성 스킬

## 개요

이 스킬은 **Kling.ai 웹사이트에서 프롬프트 기반 AI 영상 자동 생성**을 담당합니다. 사용자가 제공한 프롬프트를 Python + Selenium으로 자동 입력하고, 각 영상의 생성 완료를 대기한 후 다음 프롬프트로 진행합니다.

## 핵심 기능

### 1. 프롬프트 입력 자동화

**Kling.ai 웹 UI 제어:**
```
1. Chrome 브라우저 자동 시작
2. https://kling.ai/ 접속
3. 사용자 로그인 대기 (수동 로그인)
4. 프롬프트 입력 필드 찾기
5. 프롬프트 텍스트 입력
6. "생성" 또는 "Create" 버튼 클릭
```

### 2. 생성 완료 모니터링

**상태 확인:**
```
- 생성 중: "Processing...", "Generating..." 표시 모니터링
- 생성 완료: 영상 미리보기 또는 다운로드 버튼 표시 감지
- 에러 처리: 실패 메시지 감지 및 재시도
```

**타임아웃 설정:**
- 기본: 10분 (600초)
- 초과 시: 해당 프롬프트 건너뛰고 다음으로 진행

### 3. 순차 처리

**처리 흐름:**
```
프롬프트 1 입력 → 생성 시작 → 완료 대기 → 완료 확인
         ↓
프롬프트 2 입력 → 생성 시작 → 완료 대기 → 완료 확인
         ↓
... (반복)
         ↓
모든 프롬프트 완료 → 최종 보고
```

### 4. 실시간 상태 보고

**사용자에게 실시간 업데이트:**
```
[1/20] 프롬프트 1 생성 중... (경과 시간: 1분 30초)
[2/20] 프롬프트 1 생성 완료 ✓ (소요 시간: 2분 45초)
[2/20] 프롬프트 2 생성 중... (경과 시간: 0분 30초)
```

## 사용 시나리오

### 기본 사용

**입력:**
```
"이 20개 프롬프트로 Kling.ai에서 영상 생성해줘"
[20개 프롬프트 제공]
```

**동작:**
1. Chrome 브라우저 자동 시작
2. Kling.ai 접속
3. 사용자: "수동으로 로그인해주세요"
4. 사용자가 Kling.ai에 로그인 (Selenium 자동화는 대기)
5. 프롬프트 1 자동 입력 & 생성 시작
6. 생성 완료 대기 (약 2~3분)
7. 완료 후 프롬프트 2 진행...
8. 모든 프롬프트 처리 완료

**출력:**
```
✅ 영상 생성 완료!

[결과]
- 프롬프트 1: ✓ (2분 45초)
- 프롬프트 2: ✓ (2분 30초)
- ... (총 20개)

총 소요 시간: 50분 30초
생성된 영상: Kling.ai 내 내 작업 폴더에 저장됨
```

## 주의사항

### DO ✅

✅ 각 프롬프트마다 충분한 대기 시간 할당 (Kling.ai 생성 시간: 2~5분)  
✅ 네트워크 오류 시 자동 재시도 (최대 3회)  
✅ 사용자에게 실시간 진행 상황 보고  
✅ 생성 중 브라우저 창 닫기 방지  

### DON'T ❌

❌ 너무 짧은 타임아웃 설정 (최소 10분)  
❌ 로그인 없이 프롬프트 입력 시도  
❌ 여러 탭에서 동시 생성 (순차 처리만)  

## 에러 처리

| 에러 | 원인 | 대응 |
|------|------|------|
| 로그인 필요 | 세션 만료 | "로그인해주세요" 메시지 & 대기 |
| 네트워크 오류 | 연결 끊김 | 자동 재시도 (3회) |
| 타임아웃 | 생성 10분 초과 | 해당 프롬프트 건너뛰고 다음 진행 |
| 프롬프트 입력 실패 | UI 변경 | 에러 보고 후 수동 개입 요청 |

## 테스트 시나리오

### 테스트 1: 단일 프롬프트
입력: 1개 프롬프트
기대: 브라우저 시작 → 로그인 대기 → 프롬프트 입력 → 생성 완료

### 테스트 2: 다중 프롬프트
입력: 5개 프롬프트
기대: 5개 모두 순차 처리, 각각 생성 완료 확인

### 테스트 3: 에러 처리
입력: 프롬프트 중 로그인 만료
기대: 자동 재로그인 요청 후 진행

## 보안 주의사항

- ⚠️ Kling.ai 크리덴셜을 저장하지 않음 (사용자가 수동으로 로그인)
- ⚠️ 브라우저는 HeadlessBrowser 모드 미사용 (사용자가 모니터링 가능)
- ⚠️ 자동화 감지 회피 로직 포함 (user-agent 변경 등)
