# 🎬 인어왕자 20개 프롬프트로 Kling.ai 영상 생성하기

## 📋 개요

이 가이드는 이전에 생성한 **인어왕자의 운명: 사랑과 선택 - 20개 프롬프트**를 Kling.ai에서 자동으로 입력하여 영상을 생성하는 방법을 설명합니다.

---

## 🚀 빠른 시작 (3분 안에)

### 1단계: 필요한 패키지 설치 (PC에서)

```bash
pip install selenium webdriver-manager
```

### 2단계: 스크립트 다운로드

아래 파일을 다운로드하여 당신의 PC에 저장합니다:
```
scripts/run_kling_20_prompts.py
```

### 3단계: 스크립트 실행

```bash
python run_kling_20_prompts.py
```

또는

```bash
python3 run_kling_20_prompts.py
```

### 4단계: 로그인 및 자동 생성

스크립트가 실행되면:
1. Chrome 브라우저가 자동으로 Kling.ai로 이동
2. 당신이 수동으로 로그인
3. 터미널에서 엔터 누르기
4. 스크립트가 프롬프트 20개를 순차적으로 복사 및 생성 안내

## 🎯 사용 방법 (상세)

### 1. 스크립트 시작

```bash
python3 run_kling_20_prompts.py
```

**출력:**
```
================================================================================
🎬 인어왕자의 운명: 사랑과 선택 - Kling.ai 자동 생성
================================================================================

📝 총 20개의 프롬프트를 순차적으로 생성합니다.
⏱️  예상 소요 시간: 약 60~100분 (1시간)

🌐 Chrome 브라우저를 시작합니다...

✅ Kling.ai 페이지가 열렸습니다.
```

### 2. Kling.ai 로그인

Chrome 브라우저가 자동으로 https://kling.ai로 이동합니다.

**당신이 할 일:**
- Kling.ai에 로그인 (이메일, Google, 기타 방식)
- 로그인 완료 후 터미널로 돌아가기

```
⏳ 로그인을 완료했으면 엔터를 누르세요: [여기서 엔터 누르기]
```

### 3. 프롬프트 자동 처리

로그인 후 스크립트가 각 프롬프트를 순차적으로 처리합니다:

```
================================================================================
[1/20] EP1 - 씬 1: 정혼식 준비
================================================================================

📝 프롬프트 (길이: 250자):

반짝이는 산호와 보석으로 장식된 화려한 인어 궁전의 광장. 푸른 물빛이 흘러들어오는...

👉 위 프롬프트를 Kling.ai에 입력하고 생성을 시작해주세요.
   (텍스트 전체를 복사하려면 아래 명령을 참고하세요)

📋 전체 프롬프트 (복사용):
반짝이는 산호와 보석으로 장식된 화려한 인어 궁전의 광장. 푸른 물빛이 흘러들어오는 천장...
[전체 프롬프트 표시]

생성 버튼을 클릭한 후 엔터를 누르세요:
```

### 4. 각 프롬프트별 처리

**당신이 할 일:**

1. **프롬프트 텍스트 복사**
   - Kling.ai의 프롬프트 입력창에 위 텍스트를 붙여넣기

2. **생성 버튼 클릭**
   - Kling.ai에서 "Generate" 또는 "생성" 버튼 클릭

3. **터미널에서 엔터**
   - 터미널에서 엔터를 누르기
   ```
   생성 버튼을 클릭한 후 엔터를 누르세요: [엔터 누르기]
   ```

4. **영상 생성 대기**
   ```
   ⏳ 영상 생성 중... (약 2~5분 소요, 1번째 영상입니다)

   영상 생성이 완료되면 엔터를 누르세요: [완료 후 엔터]
   ```

5. **완료 확인**
   ```
   ✅ EP1 씬 1 완료!
   ```

### 5. 반복

2~20번째 프롬프트까지 4번 과정을 반복합니다.

최종 완료:
```
================================================================================
🎉 모든 프롬프트 생성이 완료되었습니다!
================================================================================

✨ 생성된 20개의 영상이 당신의 Kling.ai 대시보시에 저장되었습니다.
📥 필요하면 각 영상을 다운로드하여 편집할 수 있습니다.
```

## 💡 팁 & 트릭

### 복사 편의성

스크립트가 각 프롬프트를 전체 표시하므로:

1. 프롬프트 텍스트 전체 선택 (마우스 드래그 또는 Ctrl+A)
2. 복사 (Ctrl+C)
3. Kling.ai 입력창에 붙여넣기 (Ctrl+V)
4. 생성 버튼 클릭

### 에러 처리

만약 생성 실패:
- Kling.ai 페이지 새로고침 (F5)
- 같은 프롬프트를 다시 입력
- 터미널에서 다시 엔터

### 다른 작업 함께하기

생성 대기 시간에:
- 컴퓨터의 다른 작업 진행 가능
- 커피나 간식 준비
- 문서 검토 등

**주의:** 브라우저 창을 닫으면 안 됩니다!

## ❓ FAQ

### Q1: 브라우저를 최소화해도 돼?
**A:** 네, 브라우저를 최소화해도 됩니다. 닫기만 하지 않으면 괜찮습니다.

### Q2: 중간에 멈추면?
**A:** 스크립트를 다시 실행하고, 마지막에 완료한 프롬프트부터 다시 시작하세요. Kling.ai는 이미 생성한 영상을 중복으로 생성하지 않습니다.

### Q3: Kling.ai 크레딧 필요한가?
**A:** 네, 각 영상 생성마다 크레딧이 소비됩니다. 사전에 충분한 크레딧을 확보하세요. (보통 1개 영상 = 1~5 크레딧)

### Q4: 프롬프트 수정은?
**A:** 필요하면 Kling.ai에서 직접 프롬프트를 수정하여 다시 생성할 수 있습니다.

### Q5: 더 빨리 할 수 없나?
**A:** Kling.ai 자체 생성 시간은 최소 2~3분이므로 더 빨리 할 수 없습니다. 평행하게 진행하려면 여러 계정이 필요합니다.
