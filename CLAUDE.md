# 프로젝트: jarayo_02 - AI 영상 제작 하네스

## 🎬 하네스: AI 영상 제작

**목표:** 드라마 스토리를 입력받아 AI 영상 생성용 5부작 프롬프트 자동 생성

**트리거:** 영상 제작, 스토리 프롬프트, 드라마 시나리오 관련 요청 시 `ai-drama-orchestrator` 스킬을 사용합니다.
단순 질문이나 스토리 작성 상담은 직접 응답 가능합니다.

---

## 📊 하네스 구조

### 에이전트 (`.claude/agents/`)
| 에이전트 | 역할 | 입력 | 출력 |
|---------|------|------|------|
| **story-architect.md** | 스토리 분석 및 5부작 분해 | 사용자 드라마 스토리 | 5부작 분석 (JSON + 마크다운) |
| **prompt-generator.md** | 각 에피소드별 프롬프트 생성 | 에피소드 가이드 | 4개 프롬프트 × 5회 |
| **coordinator.md** | 전체 워크플로우 조율 | 모든 중간 산출물 | 최종 통합 문서 |

### 스킬 (`.claude/skills/`)
| 스킬 | 담당 에이전트 | 기능 |
|-----|--------------|------|
| **story-developer/SKILL.md** | story-architect | 스토리 분석, 에피소드 분해, 감정 곡선 설계 |
| **video-prompt-generator/SKILL.md** | prompt-generator | 15초 프롬프트 작성 (영상 생성 AI 최적화) |
| **orchestrator/SKILL.md** | coordinator | 전체 워크플로우 조율, 팀 관리 |

### 실행 모드
- **기본 모드:** 에이전트 팀 (story-architect + prompt-generator×5 + coordinator)
- **병렬 처리:** 5개 에피소드 프롬프트 동시 생성

---

## 🚀 사용 방법

### 기본 사용

```
"인어왕자가 정략결혼을 거부하고 인간여자와 사랑하는 막장 스토리로 
5부작 영상 프롬프트 만들어줘"
```

**자동 처리:**
1. 스토리 아키텍트 → 스토리 분석 및 5부작 분해
2. 프롬프트 크리에이터 ×5 → 각 에피소드별 4개 프롬프트 병렬 생성
3. 코디네이터 → 최종 통합 및 가이드 문서 작성

### 출력물
- ✅ 5부작 스토리 분석
- ✅ 총 20개 프롬프트 (에피소드별 4개 × 5회)
- ✅ 캐릭터 레퍼런스 시트
- ✅ 제작 가이드 및 체크리스트
