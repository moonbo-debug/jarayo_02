## 🎭 캐릭터 레퍼런스

### 주인공 1: 인어왕자

**기본 외형**
- 긴 흰색 머리 (어깨까지, 우아함)
- 밝은 파란색 비늘 (신체 전체)
- 우아한 체형 (마른 편, 왕족다운 자세)
- 밝고 아름다운 얼굴

**에피소드별 변화**
| 에피소드 | 복장 | 표정 | 상태 |
|---------|------|------|------|
| EP1 | 화려한 파란 궁전복 | 불안, 저항 | 정략결혼 강압 중 |
| EP2 | 궁전복 + 캐주얼 옷 | 행복, 설렘 | 인간여자와 비밀 만남 |
| EP3 | 소박한 어두운 옷 | 결연함, 절망 | 도망 중 |
| EP4 | 궁전복 (부상 회복 중) | 무감각, 혼란 | 기억 상실 상태 |
| EP5 | 우아한 복장 → 인간 옷 | 깨달음 → 결연함 | 기억 회복 → 인간 변신 |

**성격 진화**
- EP1~2: 민감하고 로맨틱한 왕자
- EP3: 용기 있는 선택을 하는 모습
- EP4: 기억을 잃은 공허한 상태
- EP5: 진정한 사랑을 깨닫고 결연한 모습

**특수 효과 (EP5)**
- 기억 회복 순간: 플래시백 이미지 폭주
- 변신 순간: 파란 비늘이 사라지고 인간의 다리가 나타남 (마법적 효과)

---

### 주인공 2: 인간여성

**기본 외형**
- 갈색 긴 머리 (허리까지)
- 밝은 갈색/녹색 눈
- 중간 키, 따뜻한 인상
- 수줍음이 있지만 강한 의지

**의상**
- 일관된 흰색/크림색 여름 원피스 (5부작 전체)
- 모래와 물에 자주 젖음 (자연스러운 표현)

**성격**
- 진심 있고 따뜻함
- 왕자를 진정으로 사랑함
- 기다릴 수 있는 인내심
- 절망하지 않는 희망

**에피소드별 역할**
| 에피소드 | 역할 | 상태 |
|---------|------|------|
| EP1 | 구조받는 여자, 첫 만남 | 신비로움 속 깨어남 |
| EP2 | 비밀의 사랑 | 행복, 헌신적 |
| EP3 | 기다리는 여자 | 왕자의 도망을 기다림 |
| EP4 | 혼자 남은 여자 | 절망하지 않고 기다림 |
| EP5 | 재상봉 + 새로운 시작 | 왕자를 다시 만나 함께 걸음 |

---

### 주인공 3: 다른 왕국의 인어공주

**기본 외형**
- 검은 비늘 (신체 전체)
- 검은 긴 머리 (거의 검은색)
- 위압적이고 강인한 체형
- 지배욕이 묻어나는 차가운 얼굴

**복장**
- 검은색 기본 + 빨간색 액센트의 화려한 궁전복
- 왕권을 상징하는 액세서리

**성격**
- 권력욕이 강함
- 인어왕자를 소유하고 싶어함
- 집착적이고 냉정
- 막무가내식 행동

**에피소드별 역할**
| 에피소드 | 역할 | 액션 |
|---------|------|------|
| EP1 | 정약혼자 (배경) | 등장 예정 |
| EP2 | 위협 (배경에서) | 왕자 감시 |
| EP3 | 갈등의 중심 | 강제 결혼 강행 |
| EP4 | 구출자 (역설적) | 왕자를 물에서 구함 |
| EP5 | (배경에서) | 궁전에서 기다림 |

## 🎬 전체 프롬프트 통합 (요약)

### EP1: 불운한 정혼 (60초)

**전체 감정 호:** 불안 → 저항 → 설렘 → 희망

**색감 변화:** 파란색 & 은빛 → 검은색 → 밝은 파란색 → 황금색

**음향 톤:** 불안한 현악기 → 긴박한 오케스트라 → 우아한 현악기 → 부드러운 현악기

**4개 씬:**
1. 정혼식 준비 (불안감)
2. 인간여자 구조 (영웅적 순간)
3. 왕궁 돌아옴 (거리감 & 불일치)
4. 몰래 해변 방문 (사랑의 시작)

---

### EP2: 금지된 사랑 (60초)

**전체 감정 호:** 설렘 → 깊은 사랑 → 순수한 행복 → 불안

**색감 변화:** 황금색 → 은빛 → 밝은 황금색 → 어두운 톤

**음향 톤:** 낭만적 → 감정적 → 경쾌함 → 불안함

**4개 씬:**
1. 해변에서의 비밀 만남 (로맨스)
2. 사랑하는 마음 고백 (깊은 감정)
3. 함께하는 시간의 행복 (순수함)
4. 왕국의 의심 (위협의 그림자)

---

### EP3: 강압과 도망 (60초)

**전체 감정 호:** 긴장 → 절망 → 결연함 → 공포

**색감 변화:** 검은색 & 빨간색 → 어두운 톤 → 은색 & 파란색 → 검은색

**음향 톤:** 위협적 드럼 → 드라마틱 → 희생의 현악기 → 긴급 음악

**4개 씬:**
1. 다른 공주 도착 (위협)
2. 왕국 지배자의 강압 (절망)
3. 왕자의 도망 결심 (결연함)
4. 도주 중 생명 위기 (공포)

---

### EP4: 기억의 바다 (60초)

**전체 감정 호:** 절망 → 영웅적 구출 → 혼란 → 공허함

**색감 변화:** 검은색 → 어두운 파란색 → 부드러운 파란색 → 밝은 파란색

**음향 톤:** 침묵 → 영웅적 → 신비로운 → 아름답지만 슬픈

**4개 씬:**
1. 왕자 물에 빠짐 (절망)
2. 다른 공주가 구함 (영웅적)
3. 깨어남 & 기억 상실 (혼란)
4. 새로운 공주와의 감정 (공허함 속의 새로운 사랑)

---

### EP5: 깨어난 진실 (60초)

**전체 감정 호:** 그리움 → 충격 → 갈등 → 희망

**색감 변화:** 은빛 → 빨간색 → 파란색 & 황금색 → 밝은 황금색

**음향 톤:** 그리운 현악기 → 극적 → 혼합 → 웅장한 오케스트라

**4개 씬:**
1. 인간여자 목격 (기억의 편린)
2. 기억 완전 회복 (충격)
3. 두 세계의 선택 (갈등 & 결단)
4. 인간이 되기로 결심 (희망 & 새로운 시작)

## 🎨 통합 색감 & 음향 가이드

### 색감 팔레트 (5부작 전체)

| 색상 | 의미 | 사용 에피소드 |
|------|------|--------------|
| **깊은 파란색** | 인어 세계, 신비로움, 불안 | EP1~2, EP4 초반 |
| **밝은 황금색** | 인간 세계, 사랑, 희망 | EP2, EP5 |
| **검은색/빨간색** | 위협, 갈등, 강압 | EP3 |

**인어 세계 BGM:** 클래식 현악기, 신비로운 음향

**갈등 순간:** 드럼, 긴박한 현악기
**감정적 순간:** 바이올린 솔로, 부드러운 음악
**클라이맥스:** 웅장한 오케스트라

## 💡 제작 팁

### 프롬프트 최적화

**Runway AI의 경우:**
- "A 15-second cinematic video of..."로 시작
- 구체적인 카메라 움직임 지시 (pan, zoom, slow motion 등)
- 색감은 RGB 값보다 감정적 표현 사용 ("warm golden light" vs "#FFD700")

**Gen-3의 경우:**
- "4K cinematic quality"로 명시
- 간결한 명사 중심 표현
- 배경 디테일 강조

**Pika의 경우:**
- 중간 길이 프롬프트 (150~200자)
- 감정과 움직임 균형 있게
- 특수효과는 구체적으로 명시

### 캐릭터 일관성 유지

1. **첫 생성 시 스타일 정의:**
   - 첫 프롬프트(1-1)를 최대한 상세하게 작성
   - 생성된 영상을 기준으로 나머지 프롬프트 조정

2. **캐릭터 스타일 문서:**
   - 각 EP 시작 시 이전 생성 이미지 참고
   - 일관된 외형 유지를 위해 프롬프트 수정

3. **배경 지속성:**
   - 인어 궁전은 항상 산호 + 보석으로
   - 해변은 항상 모래 + 파도로 표현
